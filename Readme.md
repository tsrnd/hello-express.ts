# Hello-node.ts (Team2)

### How To Run This Project (for development)
- Download dependencies

```
cd hello-node.ts
npm install
```

- Build and Run
```
docker-compose up --build
```

- Check TSLint(TS convention)
```
npm run tslint
```

- Fix error TSLint (fix some error TS convention)
```
npm run tslint-fix
