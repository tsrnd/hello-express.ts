# Hello-node.ts (Team4)

### How To Run This Project (for development)

- Get project by git clone
```
git clone..
```

- Download dependencies

```
cd hello-node.ts
npm install
```

- Build and Run
```
docker-compose up --build -d
```

- Fresh logs
```
docker logs -f node_app
```
